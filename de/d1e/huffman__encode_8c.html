Detailed Description

Huffman encoding of given words.

It takes 3 steps to make encoded number array.

1: make linked list of characters given. 2: make tree inside the list with huffman algorithm. Head node of the list will be the top of the tree when its done. 3: travel the tree and at the same time, make char arrays of encoded numbers.
